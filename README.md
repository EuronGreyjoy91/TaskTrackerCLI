## TaskTracker CLI

### Description
```
Task Tracker CLI using Java 21 for roadmap.sh
```
### Installation

```
Just clone the repo, run it, and type one of the following commands :)
```

### Available commands

```
- add [description]
- update [id]
- delete [id]
- mark-in-progress [id]
- mark-done [id]
- list
- list-done
- list-todo
- list-in-progress
- help
- exit
```
